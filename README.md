# 🏎️ Formula 1 Website

## Overview
This project is a simple HTML and CSS website that introduces **Formula 1**, the highest class of international single-seater motor racing. The website provides an overview of the sport, the structure of Formula 1 cars, their history, and highlights some of the top Formula 1 drivers of 2025.

---

## Features

- Modern landing page with Formula 1 theme
- Embedded Formula 1 introduction video
- Introduction to Formula 1
- Formula 1 characteristics
- Structure of Formula 1 cars
- History of Formula 1 cars
- Top Formula 1 drivers of 2025 with clickable profile links
- Image hover animations
- Responsive hover effects on headings and images
- Dark-themed user interface

## How to Run

1. Download or clone this project.
2. Keep all images and video files in the same folder as the HTML file.
3. Open **FORMULA 1.HTML** in any modern web browser.

No additional software or installation is required.
